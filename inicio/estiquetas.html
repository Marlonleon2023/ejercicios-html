Etiquetas HTML

Section

Una "sección" es un elemento estructural utilizado para dividir y organizar el contenido de una página web. La etiqueta section se utiliza para definir una sección independiente o agrupación de contenido relacionado en un documento HTML.

Body

En HTML, body es un elemento que representa el contenido principal de un documento HTML. Es dentro de la etiqueta body donde se coloca todo el contenido visible de una página web, incluyendo texto, imágenes, enlaces, videos, formularios y cualquier otro elemento que se desee mostrar en el navegador web.

H1

La etiqueta h1 se utiliza para definir el encabezado principal de una página web. Es uno de los seis niveles de encabezado disponibles en HTML, que van desde h1 (el más importante) hasta h6 (el menos importante).
