Glosario de Términos HTML

<html>

El elemento HTML representa la raíz de un documento HTML. Todos los demás elementos HTML deben ser hijos de este elemento.

<body>

El elemento body contiene todo el contenido de un documento HTML que no está en el encabezado (head) o en un pie de página (footer).

<section>

El elemento section define una sección en un documento HTML. Esencialmente, agrupa contenido relacionado.

<div>

El elemento div se utiliza como un contenedor genérico para elementos HTML. No tiene ningún significado específico.

<p>

El elemento p define un párrafo en un documento HTML.

<a>

El elemento a define un hipervínculo, que se utiliza para enlazar de una página a otra.

<img>

El elemento img se utiliza para incrustar imágenes en un documento HTML.

<strong>

El elemento strong se utiliza para dar énfasis a un texto, lo que generalmente se muestra en negrita.

<em>

El elemento em se utiliza para enfatizar texto, que generalmente se muestra en cursiva.

<span>

El elemento span se utiliza para aplicar estilos a porciones específicas de texto en línea.
