Etiqueta Atributo Funcion
Section id: Un identificador único para el elemento. class: Una clase que sirve para estilizar el elemento con CSS o identificarlo para manipulación mediante JavaScript. style: Reglas de estilo en línea para aplicar al elemento. sta etiqueta permite agrupar contenido lógico en secciones separadas dentro de un documento, lo que facilita la organización y la estructuración de contenido.
